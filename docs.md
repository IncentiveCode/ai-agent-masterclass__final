# content pilot

## 이름

- Content Pilot
- 사용자가 콘텐츠를 입력하면, 최종 목적지까지 안내한다는 뜻을 가지고 있습니다.

## 목적

- Google Sheets에 등록된 원고를 자동으로 교정 & 변환하여 블로그 / SNS / PPT 등 각 형식에 맞는 콘텐츠로 제작합니다.
- 일종의 멀티채널 콘텐츠 제작 파이프라인 에이전트입니다.

## 핵심 기능

- 도구 활용 : 단순 LLM 프롬프트가 아닌, 맞춤법 검사 / 트렌드 검색 / 스타일 가이드 참조 등 도구를 자율적으로 선택해서 교정합니다.
- 셀프 리뷰 : 에이전트가 교정 결과를 자동 평가하고, 기준 미달시 자동 재교정합니다. 사람에게 평가 받기 전에 품질을 끌어올릴 수 있습니다.
- Human-in-the-loop : 교정 결과를 사람이 확인하고 추가로 수정 요청할 수 있습니다.
- 멀티 포맷 출력 : 블로그 포스팅, SNS 카드 뉴스, PPT 슬라이드 등 원하는 형태에 맞춰 결과물을 출력합니다.
